◆ hashmap_t

typedef struct hashmap_t hashmap_t

Handle to a hashmap.

Quick access of existing addresses to check for leaks or double free

Remarks
Exists on each heap and is destroyed just before the heap
Is directly alloced memory; hueristically protected from running out of memory by being reliant on heap's available memory
